# Mooto'z Bakery

100% eggless bakery website for Mooto'z Bakery, Kota, Rajasthan.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

- `/` — Home
- `/privacy-policy` — Privacy Policy
- `/terms-and-conditions` — Terms & Conditions
